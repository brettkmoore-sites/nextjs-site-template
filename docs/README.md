# /docs

Operational notes for running the site. NEVER secrets — use Vercel env vars for those.

## Suggested files

- `GBP-SETUP.md` — Google Business Profile setup checklist
- `GBP-MAINTENANCE.md` — weekly post cadence + review-request scripts
- `NAP-CONSISTENCY.md` — canonical NAP block + checklist of citation properties
- `<integration>-WEBHOOK-SETUP.md` — per-integration setup (KVCore, Google Sheets, Sendgrid, etc.)
- `VENDOR-LOGINS.md` — list of vendor admin accounts (link only, no credentials)
- `CMS-HANDOFF.md` — if/when a CMS is added, the handoff notes for the client
