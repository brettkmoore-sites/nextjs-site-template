# /content/brief

Per-page content briefs in plain English. Components in `/src/app/` read these. Non-developer collaborators can edit copy here without touching React.

## Standard files (one per page on the site)

- `HERO-COPY.md` — homepage hero: 3 variants for A/B testing (StoryBrand, Dan Kennedy, the brand-archetype variant)
- `ABOUT-COPY.md` — long-form founder story
- `FOR-<persona>-COPY.md` — one per primary ICP persona (sellers, buyers, families, athletes, coaches, etc.)
- `QUIZ-DESIGN.md` — quiz scenarios, scoring tiers, result screens, lead-capture flow
- `DESIGN-DIRECTION.md` — palette, typography, photography direction, animation rules

## Optional files

- `RESOURCES-COPY.md`, `PREFERRED-VENDORS-COPY.md`, `JOIN-THE-TEAM-COPY.md` — ancillary pages
- `CONTACT-COPY.md` — contact page voice + form fields

## Page-level skeleton (every long-form page follows this)

```
HERO
THE PROBLEM (PAS — name the fear, agitate, solve)
OLD WAY vs OUR WAY
THE GRAND SLAM OFFER (Hormozi-style value stack with $ figures)
PROOF (named-client case studies)
SOCIAL PROOF (testimonial pulls)
LEAD MAGNET (free PDF / checklist / worksheet)
FINAL CTA (repeat primary CTA + tel: link)
```
