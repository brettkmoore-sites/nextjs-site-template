# /content/testimonials

Raw client quotes, attributed. One file per client. Components import these to render testimonial walls + page-specific testimonial pulls.

## Standard frontmatter

```yaml
---
client_name: "<First Last>"
client_role: "<role/title — optional>"
city: "<City, ST>"
date: 2026-04-12
source: "google-review | direct-quote | email | text"
verified: true
star_rating: 5
---
```

Body is the verbatim quote.
